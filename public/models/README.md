# DNA Model

Place your DNA helix GLTF/GLB model here as `dna.glb`.

**Recommended source**: [Sketchfab Human DNA](https://sketchfab.com/3d-models/human-dna-746f0754e7cb45a4b54840e1b858b01d)

**File path**: `public/models/dna.glb`

**Optimization tips**:
- If polycount is high, use Blender or glTF-Pipeline to decimate
- Target: ~50k-100k triangles for smooth performance
- Ensure materials are PBR-compatible
